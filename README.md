# InAmigos Foundation - Awareness Page

A clean, responsive, and professional landing page UI designed for the **InAmigos Foundation**. This project highlights the NGO's core initiatives, specifically focusing on "Uniting Minds For Change" through community-driven initiatives like **Project Vikas** and **Project Udaan**. 

Built with semantic HTML5 and pure CSS3, this template is lightweight, easy to customize, and ready to be integrated into any backend or content management system.

---

## Features

* **Modern & Clean UI:** Designed with a professional, trustworthy color palette suitable for non-profit organizations.
* **Hero Section:** Prominent display of the organization's motto and immediate Calls-To-Action (CTAs).
* **Project Highlight Cards:** Dedicated, visually appealing cards for ongoing projects (Project Vikas & Project Udaan).
* **Impact Statistics:** A clean statistics banner to showcase real-world impact (meals served, students reached, volunteers).
* **No Framework Dependencies:** Built entirely with vanilla HTML and CSS for maximum performance and easy maintenance.
* **Iconography:** Integrated with FontAwesome for scalable, lightweight vector icons.

---

## Technologies Used

* **HTML5:** Semantic structure and accessible layout.
* **CSS3:** Custom styling, Flexbox for layout management, and smooth hover transitions.
* **FontAwesome (v6.0.0):** Used for UI icons.
